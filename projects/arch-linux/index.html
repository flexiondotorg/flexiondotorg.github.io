Arch Linux

Arch Linux AUR MATE Desktop pacman PKGBUILD systemd

Arch Linux is a Linux distribution that uses a rolling release model, such that a regular system update is all that is needed to obtain all the latest software. The design approach follows the KISS principle (“keep it simple, stupid”) and focuses on elegance, code correctness, minimalism and simplicity.
